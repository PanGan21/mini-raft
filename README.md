# mini-raft
Minimal and simple key/value raft implementation in go for learning

## Run demo servers
terminal 1: `make server-1`
terminal 2: `make server-2`
terminal 3: `make server-3`
terminal 4: `make server-4`

## Run demo client
terminal 5: `make client-1`

## Example Commands
`SET "example-key" 1`

`GET "example-key"`
